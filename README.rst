Installation
============

http://luarocks.org/modules/dryproject/drylib

To install the library locally, execute::

   $ luarocks install drylib

To uninstall the library, execute::

   $ luarocks remove drylib

Usage
=====

To load the library, do::

   dry = require 'drylib'
